When to use each? Medium Article

Use if/else when:

  • Dealing with complex conditions with multiple branches
  • Needing to execute more than 1 statement
  • Clarity is essential

Use ternary when:

  • Simple conditions that require 1 line actions
  • Concise and clean code

Functions in Javascript

Use functions to wrap logic and make it reusable. They are great for avoiding mutations.

Javascript has 2 ways of declaring functions: traditional notation and arrow notation.

Traditional Notation

function calculateSomething() { // enter code here }

Arrow Notation

const calculateSomething = () => { // enter code here }

Function Syntax Specifics

  • The return statement is the output value of a function. It's not required but is a good idea to actually return something.
  • Function parameters are the variables that you set up in the parentheses (). These are then referred to inside the function body. They make the function reusable.
  • Function arguments are the values you pass in when you call the function. They fill in the parameters.
